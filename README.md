#  Alien Sky


**Alien Sky** — это аркадная игра на Python с использованием библиотеки Pygame. Игрок управляет космическим кораблём, стреляет по пришельцам, проходит уровни и зарабатывает очки.

---

##  Особенности игры

-  4 уровня сложности
-  Цепная реакция от розовых пришельцев
-  Жёлтые пришельцы могут стрелять
-  Синие пришельцы меняют размер
-  Магазин улучшений между уровнями
-  Система рекордов с сохранением
-  Настройка громкости звуков

##  Как установить и запустить

### 1. Установите Python

Убедитесь, что у вас установлен Python 3.8 или выше. Скачать можно с [официального сайта](https://www.python.org/downloads/).

### 2. Скачайте код

**Способ 1: Через GitHub**
```
git clone https://github.com/mzamyko/student-practice.git
cd student-practice
```

**Способ 2: ZIP-архивом**
- На странице репозитория нажмите **Code → Download ZIP**
- Распакуйте архив

### 3. Установите зависимости

Откройте командную строку (cmd) в папке с игрой и выполните:
```
pip install -r requirements.txt
```

### 4. Запустите игру
```
python alien_invasion.py
```
